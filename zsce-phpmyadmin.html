PHPmyadmin for ZendServer-CE

  • Make sure MySQL is installed (if you want use local server)
  • Make sure mysql listen to tcp port (normally 3306) and add mysqld to your /etc/hosts.allow file or use socket instead
  • Zend Server apache is configured and ready but you should config phpmyadmin by yourself
  • Create config.inc.php inside phpmyadmin base on config.sample.inc.php (make sure setup your server base on socket if you add skip-networking on my.conf )
  • Restart ZendServer (/etc/rc.d/zendserverce restart)
  • Go to http://localhost/phpmyadmin
